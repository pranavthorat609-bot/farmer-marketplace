# Farmer Marketplace Project

This is my mini project based on **Direct Farmer to Customer Marketplace**.
In this project, farmers can add their products and customers can buy them directly.

---

## Features

* Seller can add products (name and price)
* Buyer can see all products
* Buyer can place order
* Seller can see orders
* Seller can Accept or Reject orders
* Buyer can see order status (pending / accepted / rejected)

---

## Technology Used

* HTML
* CSS (Tailwind CSS)
* JavaScript
* Firebase Realtime Database

---

## How to Run

1. Download or clone the project
2. Open folder in VS Code
3. Run with Live Server

## How Project Works

* Seller adds product → stored in Firebase
* Buyer opens site → sees products
* Buyer clicks buy → order is stored
* Seller checks order → Accept or Reject
* Buyer can see updated status

---

## Project Files

* index.html → Home page
* seller-panel.html → Seller side
* buyer-panel.html → Buyer side
* app.js → Main logic
